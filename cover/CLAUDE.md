# cover/ — генерация сопроводительных писем

**Назначение.** Короткий персональный текст под конкретную вакансию + `cv.md`.

**Ключевые файлы.**
- `generator.py` — `generate_cover_letter(vacancy, profile, config)` → `CoverLetter`.

**Тон/длина/язык — из конфига** (`config.cover_letter`):
- `tone` (friendly/formal/youthful), `youthful`, `max_sentences`, `language` (ru/en/auto).
- По умолчанию: дружелюбный, человечный, молодёжный, RU; для англоязычных вакансий — EN (auto).

**Инварианты / подводные камни.**
- ⚠️ Письмо — ВСЕГДА только черновик (`CoverLetter.is_draft=True`). Ничего не отправляется
  автоматически; отклик делает пользователь руками.
- Текст должен быть заточен под конкретную вакансию и реальный CV — не шаблонная вода.
- Команда `/cover <id>` перегенерирует письмо для одной вакансии.

**Статус:** ✅ реализовано. `language: auto` определяет язык по тексту вакансии. Тесты — `tests/test_cover.py`.
