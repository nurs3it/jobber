# .claude/ — команды, агенты, документация

**Назначение.** Интерфейс продукта для Claude Code: slash-команды, субагенты и доки.

**Структура.**
- `commands/` — slash-команды (`.md` с описанием и шагами):
  `onboard`, `scan`, `digest`, `cover`, `profile-edit`, `sources`, `seen-reset`.
- `agents/` — субагенты (напр. `matcher.md` — оценка вакансий под профиль).
- `docs/` — `setup.md`, `architecture.md`, `adding-a-source.md`, `runbook.md`, `security.md`, `troubleshooting.md`.

**Как работать/расширять.**
- Команда = понятный сценарий из шагов, опирающийся на MCP-инструменты `mcp_server/` и правила.
- Логику «как анализировать резюме» / «как писать письмо» держим в командах (`onboard.md`, `cover.md`),
  а не в коде — её исполняет сам Claude Code.

**Инварианты / подводные камни.**
- Команды read-only по Telegram. Письма — черновики.
- Держать команды и доки актуальными после каждой фазы (правило корневого `CLAUDE.md`).
