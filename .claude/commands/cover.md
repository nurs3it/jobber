# /cover <id> — письмо под вакансию

Сгенерировать или переписать сопроводительное письмо для вакансии с идентификатором `$ARGUMENTS`.

## Шаги

1. Найти вакансию по id (из последнего дайджеста/скана, кэш `storage/`).
2. Взять `profile/cv.md` и текст вакансии.
3. Написать короткий персональный текст под **эту конкретную** вакансию:
   - тон/длина/язык — из `config.cover_letter` (по умолчанию дружелюбный, человечный, молодёжный, RU;
     для англоязычной вакансии — EN при `language: auto`);
   - человечно, без шаблонной воды; зацепить релевантным опытом из CV;
   - уложиться в `max_sentences`.
4. Показать письмо пользователю. Принять правки и переписать при необходимости.

> ⚠️ Это ВСЕГДА только черновик. Отклик пользователь отправляет сам, руками.
