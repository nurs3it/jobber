# /profile-edit — правка профиля

Помочь пользователю обновить `profile/cv.md` и связанные фильтры.

## Шаги

1. Показать текущий `profile/cv.md` (структуру/ключевые поля).
2. Спросить, что изменить (роль, навыки, грейд, зарплата, локация, удалёнка, релокация, стоп-факторы, опыт).
3. Внести правки в `cv.md`, **не теряя остальные факты** (правило полноты из `/onboard`).
4. Синхронизировать соответствующие поля в `config.yaml → filters`.
5. Подтвердить итог пользователю.

> Профиль влияет на скоринг и письма — изменения применятся при следующем `/scan` / `/digest`.
